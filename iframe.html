I AM AN IFRAME

<a href="./index.html" target="_parent"> Welcome to Google Search </a>

RESULT: Welcome to Index page

Target: _parent : This opens an anchor in the same frame

<a href="./index.html" target="_self"> Welcome to Index page </a>

RESULT: Welcome to index page

target: self : This opens an anchor in the same frame,

<a href="./index.html" target="_top"> Welcome to index page </a>

RESULT: Welcome to Index page

target: top: This opens an anchor in the same frame,

<a href="./index.html" target="_blank"> Welcome to Index page </a>

RESULT: Welcome to index page

target: blank: This opens an anchor in the same frame,

<a href="./index.html" target="iframe_a"> Welcome to Google Search </a>

RESULT: Welcome to Index page

target: blank: This opens an anchor in the same frame,

For a non iframe page, property _parent, _self and _top work the same, while _blank opens page in new window or tab

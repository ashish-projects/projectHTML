Type of links in html

1.Simple and most common link

<a href="http://www.google.com/" > Welcome to Google Search </a>

RESULT: Welcome to Google Search

Default link opens in the same window, this link takes to different website

2.Local link

<a href="index.html" > index page of this website </a>

RESULT: Welcome to index page

This link opens in the same window, this link takes to same page of website

3.link with new tab as target

<a href="http://www.google.com/" target="_blank"> Welcome to Google Search </a>

RESULT: Welcome to Google Search

This link opens in the new tab

4.link with same frame as target, which is deafult

<a href="http://www.google.com/" target="_self"> Welcome to Google Search </a>

RESULT: Welcome to Google Search

This opens an anchor in the same frame, for more info

5.Images as links

<a href="http://www.google.com/" target="_blank"><img src="./images/google.png height="100px" width="100px" /> </a>

RESULT:

This image link opens in the new tab

6.link as bookmarks

<a href="#bmark">top of page </a>
RESULT: top of page

This link takes you to any bookmark on the page
